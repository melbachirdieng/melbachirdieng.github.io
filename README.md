## 🔄 Personnalisation

### Modifier les couleurs
Éditer les variables CSS dans `style.css` :
```css
:root {
    --primary: #00ff88;      /* Vert néon */
    --secondary: #00d4ff;    /* Bleu cyan */
    --accent: #ff0080;       /* Rose */
    --dark: #0a0e27;         /* Fond sombre */
}
```

### Ajouter un projet
1. Créer un dossier avec les images du projet
2. Ajouter le PDF dans `/docs/`
3. Modifier `index.html` pour ajouter une carte de projet

### Ajouter une certification
1. Ajouter le PDF dans `/Certifications/`
2. Ajouter une carte de certification dans `index.html`

## 📱 Responsive

- **Tablette** : Grilles ajustées
- **Mobile** : Navigation masquée, grilles en 1 colonne
